데이터 정리 기준

  • 유형은 대문자( ex - SA, DA) 기기 구분은 소문자(ex - pc, mobile)
  • nice는 매체 보고서가 weekly 로 오기 때문에 주중에는 기타매체에 666,667 원으로 업로드 하도록 하고, 매주 랩사에서 보고서가 오면 파일 그대로 업로드 하여 주중에 기타매체로 업로드한 값을 대체합니다.
  • First Click Conversion 데이터는 매일 9시 30분에 설정되어 10분정도 진행되므로, 안전하게 10시쯤 파일 업로드 하는것을 권장합니다.
  • Last Click Conversion 데이터를 업로드 하는 시점에 First Click Conversion 데이터를 가져오므로, 시간상의 문제로 데이터가 이상한 경우 Last Click Conversion 데이터를 재 업로드 하시면 됩니다.
매체 데이터
  • 다음 : 브랜드 검색 관련은 파일로 업로드
    • 광고비 : PC = 13333 원 / 모바일 = 23333 원 고정
  • 나이스 : 광고비 666667 원 고정
  • 페이스북 : 컨버전을 페이스북 컨버전 기준으로 작성(투자 - 2전환, 대출 - 1전환)
  • 구글 : 투자
    • 기기 구분
      • 캠페인명에 SDC가 포함되면 모바일, PC_ 가 포함되면 PC 아니면 모바일
    • 광고 구분
      • 캠페인명에 유튜브(유투브)가 있으면 '유튜브', 기기 구분은 '통합'으로 변경
      • 캠페인명에 앞 두자리가 DA 면 매체명은 gdn, 광고타입은 DA
      • 캠페인명에 앞 두자리가 SA 명 매체명은 google, 광고 타입은 SA
    • 기기 구분
      • 캠페인명에 SDC가 포함되면 모바일, PC_ 가 포함되면 PC 아니면 모바일
    • 광고 구분
      • 캠페인명에 앞 두자리가 DA 면 매체명은 gdn, 광고타입은 DA
      • 캠페인명에 앞 두자리가 SA 명 매체명은 google, 광고 타입은 SA
      • 캠페인명에 유튜브(유투브)가 있으면 매체명은 google, 광고 타입은 '유튜브'
  • 트위터 : 캠페인 이름에 동영상이 있으면 광고타입은 동영상, 아니면 DA
    • 기기 구분 : 통합
    • 상품 구분은 : 캠페인 이름의 앞 두글자

Source : 대문자 데이터 제외

Source / 매체 :
  • Source가 "buzzscreen" 인 경우 : Source = "buzzad"
  • 1의 경우가 아닌 경우 보고서 파일의 Source명 유지
유형
  • Source에 "neoclick" 이 포함되고, Medium 이 cpc 또는 cpa 인 경우ㅠ : 유형 = "SA"
  • Source 에 "brand" 가 포함된 경우 : 유형 = "BS"
  • Source 에 "power" 가 포함되고, Medium 이 cpc 또는 blog 인 경우 : 유형 = "파컨"
  • Source 에 "_avi"가 포함 되거나 Medium 이 cpv 또는 cpvc 또는 ocpm 인 경우 : 유형 = "동영상"
  • Source 에 "naver", "daum"이 포함되고, Medium 이 "cpc"인 경우 : 유형 = "SA"
  • 1~5 의 경우가 아닌 경우 : "DA"
기기
  • Source 가 "nice_event" 이고, 캠페인명이 "nice_event_m"이 아닌 경우 : 기기 = "pc"
  • Source, Medium 에 "happyscr"이 포함된 경우 : 기기 = "mobile"
  • Source, Medium 에 "whowho"이 포함된 경우 : 기기 = "mobile"
  • Source, Medium 에 "Neoclick"이 포함된 경우 : 기기 = "pc"
  • Source가 "moneta"인 경우 : 기기 = "pc"
  • Source, Medium 에 "appier"이 포함된 경우 : 기기 = "mobile"
  • Source, Medium 에 "_pc"이 포함된 경우 : 기기 = "pc"
  • Source, Medium 에 "_m"이 포함되거나
    Source가 "cashwalk" 이거나,
    Source가 "buzzscreen" 이거나,
    Source가 "buzzscreen_view" 이거나,
    Source가 "adpacker" 이거나,
    Source가 "tnk" 이거나,
    Source가 "cashslide" 이거나,
    Source가 "blind" 이거나,
    Source가 "samsungpay" 이거나,
    캠페인명이 "nice_event_m" 인 경우 : 기기 = "mobile"
  • 캠페인에 "dbxnqm"가 포함된 경우 : 기기 = "통합"

  • Source, Medium 에 "nice"가 포함된 경우 : 상품 = "대출"

  • 캠페인명에 '유튜브(유투브)'가 포함된 경우 : 유튜브
